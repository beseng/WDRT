This folder is for test and development of a new code for calculating the MLER of WEC devices.  This should allow for more convenient checks of the code during development.

The following data files are used for testing and validating purposes for the new MLER code (used in the OMAE paper):

   OMAE2016--update20160205/RegularWaves_PitchTranslationCG/RAO_heave_RM3float.dat

   OMAE2016--update20160205/RegularWaves_PitchTranslationCG/RAO_pitch_RM3float.dat
   OMAE2016--update20160205/RegularWaves_PitchTranslationCG/RAO_surge.mp4
   OMAE2016--update20160205/RegularWaves_PitchTranslationCG/RAO_surge_RM3float.dat
